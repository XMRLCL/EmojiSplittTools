# 表情包分割工具

一个简单易用的表情包网格分割工具，可以将多个表情包组合在一起的网格图片分割成单独的表情包。

## 功能特点

- 🖼️ 支持多种图片格式（PNG、JPG、GIF、BMP、WEBP）
- 🔍 自动检测网格布局
- ✂️ 自定义行列数分割
- 📁 自定义输出文件夹和文件名前缀
- 💾 支持多种输出格式（PNG、JPG、GIF、WEBP）
- 📊 实时进度显示

## 使用方法

### 方法一：直接运行 Python 脚本

```bash
python emoji_splitter.py
```

### 方法二：使用打包好的 EXE 文件

双击运行 `dist/表情包分割工具.exe`

## 操作步骤

1. **选择图片** - 点击"浏览..."按钮选择需要分割的网格表情包图片
2. **设置网格** - 输入行数和列数，或点击"自动检测网格"让工具自动识别
3. **输出设置** - 选择输出文件夹、文件名前缀和输出格式
4. **开始分割** - 点击绿色的"开始分割"按钮
5. **查看结果** - 分割完成后可选择打开输出文件夹查看结果

## 示例

假设你有一张 2行 x 3列 的表情包图片：

```
┌───┬───┬───┐
│ 1 │ 2 │ 3 │
├───┼───┼───┤
│ 4 │ 5 │ 6 │
└───┴───┴───┘
```

设置行数为 2，列数为 3，工具会将其分割成 6 个单独的表情包文件。

## 依赖

- Python 3.6+
- Pillow (PIL)
- tkinter (Python 内置)
- numpy (可选，用于自动检测)

## 安装依赖

```bash
pip install Pillow numpy
```

## 打包为 EXE

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "表情包分割工具" emoji_splitter.py
```

打包后的文件位于 `dist/` 文件夹中。
